🤖 Arduino & Robotics

The Hardware

An Arduino is a microcontroller—a tiny computer that can read sensors and control motors. It bridges the gap between software code and the physical world.

Basic Components

  • LED: A light emitting diode.
  • Resistor: Limits the flow of electricity to prevent burning out components.
  • Breadboard: A board for prototyping circuits without soldering.

The "Blink" Sketch

This is the "Hello World" of Arduino. It turns an LED on and off.


void loop() {
  digitalWrite(LED_BUILTIN, HIGH); // Turn on
  delay(1000);                     // Wait
  digitalWrite(LED_BUILTIN, LOW);  // Turn off
  delay(1000);                     // Wait
}
