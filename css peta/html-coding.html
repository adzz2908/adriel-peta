🌐 HTML Coding

The Skeleton of the Web

HTML stands for HyperText Markup Language. It isn't a programming language; it's a markup language that tells the browser how to structure content.

Common Tags

  • <h1> - Main Heading
  • <p> - Paragraph
  • <a> - Anchor (Link)
  • <img> - Image
  • <div> - Divider (Container)

Structure Example

Every HTML page needs this basic structure:


<!DOCTYPE html>
<html>
  <body>
     <h1>My Website</h1>
  </body>
</html>
