⚙️ C++ Programming

Why C++?

C++ is a high-performance language used in operating systems, browsers, and AAA games. It gives you direct control over hardware memory.

Variables

You must declare the "type" of data a variable holds:

  • int - Whole numbers (e.g., 5, 100)
  • double - Decimal numbers (e.g., 5.99)
  • char - Single characters (e.g., 'A')
  • string - Text (e.g., "Hello")

Example Code


#include <iostream>
using namespace std;

int main() {
  cout << "Hello World!";
  return 0;
}
